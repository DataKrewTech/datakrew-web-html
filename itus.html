WHAT IS ITUS

ITUS is Datakrew's in-house developed and patened product. It is a secure wireless IoT hardware to sync IoT data with MADS over 3G/4G, WiFi, LoRa or IoT Nanosatellite, and is compatible with major industrial sensor brands over multiple protocols. It can proactively detect cyber-attacks and auto-initiate defense protocols to protect critical assets and data, even against quantum attacks.

WHY USE ITUS

DATA RELIABILITY

Through high-frequency edge analytics the ITUS Nodes continuously monitor real-time sensor data patterns, noise baselines, spikes and other signal processing metrics for data reliability.

DATA SECURITY

Quantum Secure Perimeter is a virtual fence around critical IoT sensors and data. In case of a cyber attack, the affected assets are isolated and quarantined from the rest of the network.

DATA AVAILABILITY

Hops and relay over LoRa or Wi-SUN mesh network ensures redundancy in data transmission paths to the cloud server thereby guaranting data availability even in the remotest locations.
